# Obsidian Podcast

複数のRSSフィードから記事を取得し、全文の高音質音声（mp3）を生成してObsidianに保存するツール。

## 概要

- **情報取得**: 複数のRSSフィードから未取得の最新記事を取得
- **本文抽出**: URLから記事本文を高精度抽出（技術スタック未決定）
- **音声化**: テキストをmp3に変換（技術スタック未決定）
- **Obsidian連携**: Markdown + 音声ファイルを指定ディレクトリに保存
- **自動クリーンアップ**: 一定期間経過した古いmp3を自動削除

## ステータス

技術スタック・アーキテクチャは設計対話で決定予定。
